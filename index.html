PARTICIPAÇÃO EM EVENTOS

[Instrutor] Desenvolvimento Híbrido com Ionic (2017)

O Mini-Curso de Desenvolvimento Híbrido Móvel, utilizando o IONIC, ministrado na Universidade Católica do Tocantins na Semana Tecnológica II, teve como objetivo apresentar as principais diferenças entre aplicativos móveis, híbridos e web. O foco do minicurso é demonstrar os cenários em que as aplicações híbridas podem ser melhor empregadas, além de desenvolver um pequeno aplicativo como exemplo. Utilizou o Ubuntu, o Sublime Text e o NodeJS como um ambiente de desenvolvimento, além do Ionic Framework e do Cordova para a criação do aplicativo.

[Instrutor] Desenvolvimento de aplicações web com Java e PrimeFaces (2016)

O Mini-Curso de Desenvolvimento de Aplicações Web com Java e PrimeFaces tem como objetivo apresentar os principais componentes que auxiliam no desenvolvimento de aplicações web tanto no segmento de software de código aberto quanto no software comercial. Para apresentar este curso de forma objetiva e didática, pretendemos ensinar alguns recursos lógicos de programação com o Java 8 utilizando o JSF 2.0 Web Framework, utilizando gráficos web com a biblioteca PrimeFaces e para persistência de dados em um banco de dados local MySQL com o Biblioteca JPA EclipseLink (ORM).
